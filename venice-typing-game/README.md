# 타자 서바이벌 (베네치아 스타일)

35명 동시 접속 · 40종 동물 캐릭터 랜덤 배정 · 영어 타자 서바이벌 게임입니다.
**각자 다른 네트워크(집 등)에서 원격으로 접속**하는 상황에 맞춰, 클라우드 배포를 기본 권장 방법으로 안내합니다.

## 게임 규칙
1. 닉네임 입력 → 40종 동물 캐릭터 중 하나를 랜덤 배정.
2. 대기실에서 참가자 캐릭터+닉네임이 나열됨 (최대 35명). 방장(첫 입장자)이 "게임 시작".
3. 라운드 시작 전 3-2-1 카운트다운(효과음 포함) 후 영어 단어 제시.
4. 맨 위에서 2칸 아래 위치에서 시작 → **정답이면 위로 한 칸, 오답/시간초과면 아래로 한 칸** 즉시 이동 + 효과음.
5. 라운드별 제한시간: 1-10R 10초 → 11-20R 7초 → 21-26R 5초 → 27-31R 4초 → 32-36R 3초 → 37-41R 2초 → 42R~ 1초.
6. 맨 아래 물에 닿으면 탈락. 마지막 1명이 남으면 닉네임이 크게 표시되고 전체 순위가 함께 나옵니다.
7. 게임 중 **호스트만** ESC로 일시정지/재개/재시작 가능. **누구나** 화면 좌상단 "🏆 순위" 버튼으로 게임을 멈추지 않고 현재 순위를 볼 수 있습니다.

## 추천 실행 방법 — 클라우드에 올리기 (원격 참가자에게 가장 안정적)

집에 있는 참가자들이 각자 접속하려면, 포트포워딩/공유기 설정 없이 **고정 공개 주소**를 주는 게 제일 편합니다. 무료로 시작할 수 있는 [Render](https://render.com)를 기준으로 안내합니다. (Railway, Fly.io도 방식은 비슷합니다.)

1. GitHub에 이 프로젝트 폴더를 그대로 올립니다 (레포지토리 생성 후 push).
2. [render.com](https://render.com) 가입 → **New + → Web Service** → 방금 만든 GitHub 레포 선택.
3. 설정값:
   - **Build Command**: `npm install`
   - **Start Command**: `npm start`
   - **Instance Type**: Free (인원이 많으면 유료 플랜 권장 — 무료 플랜은 일정 시간 미사용 시 슬립 상태가 됩니다)
4. 배포가 끝나면 `https://프로젝트이름.onrender.com` 같은 고정 주소가 생깁니다. 이 주소를 참가자 35명에게 공유하면 각자 집에서 브라우저로 바로 접속 가능합니다.
5. Socket.io(WebSocket)는 Render의 웹 서비스에서 별도 설정 없이 정상 동작합니다.

> 무료 플랜은 트래픽이 없으면 서버가 잠들었다가, 첫 접속 시 깨어나는 데 몇십 초 걸릴 수 있습니다. 게임 시작 5~10분 전에 미리 한 번 접속해 서버를 깨워두는 걸 추천합니다. 매번 실제로 사람들을 모아서 자주 할 계획이면, 유료 플랜(월 몇 달러)으로 올리면 이 지연이 사라집니다.

## 간단 테스트용 — 임시 터널 (ngrok)

한 번만 빠르게 테스트하고 싶다면:
```bash
npm install
npm start
```
다른 터미널에서:
```bash
npx ngrok http 3000
```
ngrok이 알려주는 `https://xxxx.ngrok-free.app` 주소를 참가자들에게 공유하면 각자 다른 네트워크에서도 접속됩니다. (무료 ngrok은 접속할 때마다 주소가 바뀌고, 동시 접속/세션 제한이 있을 수 있습니다.)

## 로컬(같은 와이파이)에서 테스트

```bash
npm install
npm start
```
내 컴퓨터: `http://localhost:3000`
같은 네트워크의 다른 사람: `http://<내 컴퓨터의 사설 IP>:3000` (Windows는 `ipconfig`, Mac은 `ifconfig`로 확인)

## 설정 변경

`server.js` 상단 값들을 수정하면 됩니다.
- `MAX_PLAYERS` : 최대 동시 접속 인원 (기본 35)
- `START_ROW` : 시작 위치 (기본 2, 맨 위에서 2칸 아래)
- `MAX_ROW` : 물(최하단) 위치 (기본 10)
- `getRoundTime(round)` : 라운드별 제한시간 규칙

단어 목록은 `words.js`, 캐릭터 목록은 `animals.js`에서 자유롭게 추가/수정 가능합니다.

