# 서바이벌 게임 (타자연습 / 정보퀴즈)

35명 동시 접속 · 40종 동물 캐릭터 랜덤 배정 · 두 개의 독립된 방(타자연습 / 정보퀴즈)을 지원하는 서바이벌 게임입니다.

## 게임 규칙
1. 닉네임 입력 + 방 선택("⌨️ 타자연습" 또는 "🧠 정보 퀴즈") → 입장 → 40종 동물 캐릭터 중 랜덤 배정.
2. 대기실은 방마다 완전히 분리되어 있습니다 (같은 서버, 다른 방 = 서로 안 보임/안 섞임). 각 방 첫 입장자가 호스트가 되어 "게임 시작" 가능.
3. 라운드 시작 전 3-2-1 카운트다운(효과음) 후 문제 제시.
   - **타자연습**: 영어 단어가 그대로 화면에 보이고, 그 단어를 정확히 입력.
   - **정보 퀴즈**: 고등학교 정보 교과(컴퓨팅시스템/데이터/네트워크/알고리즘/파이썬 제어구조/디지털윤리) 문제가 나오고, 정답을 입력. 총 30문제 (순서는 매번 무작위 셔플), 30문제를 다 쓰면 그 시점 순위로 게임 종료.
4. 맨 위에서 **3칸 아래** 위치에서 시작. 정답이면 위로 한 칸(그 라운드 **가장 먼저** 맞춘 사람은 1.5칸!) 이동, 오답/시간초과면 아래로 한 칸 즉시 이동 + 효과음.
5. 제한시간은 **5초부터 시작**해서 라운드가 진행될수록 점점 줄어듭니다 (1-10R 5초 → 11-20R 4초 → 21-26R 3초 → 27-31R 2초 → 32R~ 1초).
6. 맨 아래 물에 닿으면 탈락. 마지막 1명이 남으면(또는 정보퀴즈는 30문제 소진 시) 순위와 함께 결과 표시.
7. 게임 중 **호스트만** ESC로 일시정지/재개/재시작 가능. **누구나** 화면 좌상단 "🏆 순위" 버튼으로 게임을 멈추지 않고 실시간 순위 확인 가능.
8. 화면 **우측 하단에 작은 채팅창**이 있어 대기실/게임 중 언제든 참가자끼리 채팅 가능 (헤더 클릭하면 접고 펼 수 있음).

## 🧠 정보 퀴즈 문제 수정하기

`quiz-questions.js` 파일을 열어서 아래 형식의 배열만 고치면 됩니다. 서버 코드를 건드릴 필요 없습니다.

```js
const QUIZ_QUESTIONS = [
  { question: "대한민국의 수도는?", answer: "서울" },
  { question: "1+1=? (숫자만)", answer: "2" },
  // ... 원하는 만큼 추가/삭제/수정 가능 (꼭 20개일 필요 없음)
];
```
- `question` : 화면에 보여줄 문제
- `answer` : 참가자가 입력해야 하는 정답 (앞뒤 공백 무시, 대소문자 구분 안 함)

파일을 고친 뒤 서버를 재시작(로컬이면 `npm start` 다시, 클라우드면 재배포)하면 바로 반영됩니다.

## 실행 방법 — 클라우드 배포 (원격 참가자에게 추천)

1. 이 폴더의 파일들을 **압축 없이 그대로** GitHub 레포에 올립니다 (레포 루트에 `server.js`, `package.json`, `public/` 폴더 등이 바로 보여야 함).
2. [render.com](https://render.com) → New + → Web Service → 그 레포 연결.
3. Build Command: `npm install` / Start Command: `npm start`.
4. 배포 후 생기는 `https://프로젝트이름.onrender.com` 주소를 참가자들에게 공유.

> 무료 플랜은 오래 안 쓰면 슬립 상태가 되어 첫 접속이 느릴 수 있습니다. 게임 시작 5~10분 전에 미리 한 번 접속해 깨워두세요.

## 로컬(같은 와이파이) 테스트
```bash
npm install
npm start
```
내 컴퓨터: `http://localhost:3000`
같은 네트워크의 다른 사람: `http://<내 컴퓨터의 사설 IP>:3000`

## 설정 변경

`server.js` 상단 값들:
- `MAX_PLAYERS` : 최대 동시 접속 인원 (기본 35)
- `START_ROW` : 시작 위치 (기본 3, 맨 위에서 3칸 아래)
- `MAX_ROW` : 물(최하단) 위치 (기본 10)
- `getRoundTime(round)` : 라운드별 제한시간 규칙 (기본 5초 시작)

타자연습 단어 목록은 `words.js`, 캐릭터 목록은 `animals.js`, 퀴즈 문제는 `quiz-questions.js`에서 수정합니다.

## 폴더 구조

